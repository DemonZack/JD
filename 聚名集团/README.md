## 聚名集团｜架构师 / 研发负责人（2023.05 – 至今）

本目录记录在聚名阶段的核心工作内容，围绕 GMSSH 云桌面平台、智能体能力与应用生态三条主线，展开系统设计、技术落地与业务价值。

### 工作概览

- **产品线**：GMSSH 云桌面 / 运维平台及其周边生态（应用中心、API、智能体插件等）。
- **角色定位**：架构师 + 研发负责人，负责技术选型、架构设计、核心模块实现与团队建设。
- **组织形态**：从 0 搭建并管理约 15 人研发团队，覆盖后端、前端、客户端、测试与部分运维。

### 主要工作内容（高度概括）

- 设计并实现基于 SSH 的三层代理架构（GS/GA/GW + GH），把传统命令行运维抽象成可视化桌面与应用操作。
- 搭建 GMSSH 应用中心与一批核心运维应用（GitAdmin、VPN 管理器、站点管理、数据同步、Webhook/计划任务等），将日常运维场景沉淀为可售产品。
- 推进 Gemius 智能体能力落地，为 GMSSH 增加“对话+工具执行”的智能运维入口，覆盖知识库问答、Skill 管理、多租户与工具调用授权等能力。
- 从工程视角建设 CI/CD、监控告警、日志与审计体系，确保桌面客户端、服务端与应用生态在多环境、多系统上的稳定运行。
- 面向业务与运营侧提供解决方案与技术支撑，参与授权策略设计、SaaS/私有化交付方案制定与关键客户实施支持。

### 业务价值（整体层面）

- **对公司产品线的价值**：
  - 为聚名云服务器、域名与企服产品提供统一的运维入口，形成“买资源 + 管资源”的一体化体验；
  - 通过 GMSSH 授权、应用中心与增值服务，为公司创造新的营收增长点；
  - 降低非技术客户使用 Linux 与服务器的门槛，减少大量基础运维工单。
- **对团队与组织的价值**：
  - 建立以架构蓝图为核心的研发共识，提升团队在复杂系统下的协作效率；
  - 推动工程规范与自动化流程落地，让产品的快速迭代与稳定运行可以并行。
